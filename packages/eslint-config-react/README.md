ESLint configuration for React.

## Install

    npm install @faasjs/eslint-config-react

Then put the following code into your `package.json` file:

```json
"eslintConfig": {
  "extends": [
    "@faasjs/react"
  ]
}
```

## Usage

    npm exec eslint .
